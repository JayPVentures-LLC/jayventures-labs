# JayVentures Automation Architecture

## Brands
- jayventures → creator-facing (Wix, content, audience)
- JAYPVENTURES LLC → operations (Microsoft 365, identity, workflows)

## Principles
- Wix captures signals, not logic
- Business logic lives in Microsoft 365
- Automation is versioned and reusable
- No vendor lock-in at the workflow layer

## First Automation
Wix form submission → Microsoft 365 webhook → notify + log
