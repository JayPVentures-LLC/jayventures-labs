# Information Accessibility Theory

Status: v0.1 research draft  
Lane: jaypVLabs research, validation, and experimentation  
Owner: JayPVentures LLC  
Classification: companion research framework to ASO; experimental

## Purpose

Information Accessibility Theory (IAT) studies how information becomes available to adaptive systems, especially when information appears before conscious explanation.

IAT does not assume that every instance of intuition, insight, memory recall, deja vu, dreaming, or sudden knowing has the same mechanism. It treats each as a candidate accessibility transition requiring careful observation and competing explanations.

## Core research question

> What determines when information becomes available to an adaptive system?

## Companion relationship to ASO

Adaptive Systems Ontology (ASO) asks what functions make adaptive systems possible.

IAT asks how information moves through availability states inside those systems.

## Boundary

IAT does not prove premonition, reincarnation, extrasensory perception, or any specific unknown mechanism. It keeps unknown mechanisms available as hypotheses while requiring prospective documentation, competing explanations, and evidence-based review.

## Current strongest phrase

Information may exist, be represented, and still remain unavailable until accessibility conditions change.
