# IAT Templates

These templates support prospective documentation of information-accessibility events.

## Files

- `intuition-case-log.md` — full narrative case template.
- `intuition-case-log.csv` — structured spreadsheet-ready header template.

## Use rule

Use the Markdown template for rich detail.

Use the CSV template for structured tracking and later analysis.

Prospective cases carry more evidentiary value than retrospective cases. Do not omit misses, false positives, weak intuitions, or unresolved cases.

## Minimum viable log

At minimum, record:

- timestamp
- exact intuition
- intensity
- confidence
- predicted confirmation
- predicted disconfirmation
- action taken
- outcome status
- time to verification

## Research boundary

The templates do not assume a mechanism. They are for disciplined observation and later comparison of competing explanations.
