# Information Accessibility Theory
## Accessibility State Model v0.1

## Core distinction

Information state and accessibility state are not identical.

A system may contain, preserve, or represent information that is not currently available for conscious awareness, expression, or action.

## Draft state sequence

```text
Potential Information
  -> Available Information
  -> Represented Information
  -> Integrated Information
  -> Latent Information
  -> Accessible Information
  -> Consciously Available Information
  -> Expressed Information
  -> Applied Information
  -> Feedback
  -> Adaptation
```

## State definitions

### Potential Information

A distinguishable difference that could become information for a system but has not yet interacted with the system in an accessibility-relevant way.

### Available Information

Information present in the environment or system context that can be acquired under current conditions.

### Represented Information

Information stored or encoded in system structure.

Examples:

- memory traces
- model parameters
- notes
- symbols
- institutional records
- body-state signals

### Integrated Information

Information connected to existing representations or schemas.

### Latent Information

Represented or integrated information that is not currently accessible for deliberate use.

### Accessible Information

Information available for system use, even if not yet consciously verbalized.

### Consciously Available Information

Information available to subjective awareness or deliberate reasoning.

### Expressed Information

Information converted into language, symbol, gesture, artifact, or communicated output.

### Applied Information

Information used to guide action, decision, interpretation, or adaptation.

### Feedback

Outcome information generated by application.

### Adaptation

Change in future response, representation, accessibility, or process rule based on feedback.

## Transition classes

### Acquisition transition

Potential or available information becomes represented.

### Integration transition

Represented information becomes connected to existing models.

### Accessibility transition

Latent information becomes accessible.

### Consciousness transition

Accessible information becomes consciously available.

### Expression transition

Consciously available information becomes communicable.

### Application transition

Information changes action, judgment, or decision.

### Feedback transition

Action outcome becomes information for later processing.

## Key research target

The highest-priority unexplained transition is:

```text
Latent Information -> Accessible Information
```

This transition may be involved in:

- memory recall
- sudden insight
- intuition
- deja vu
- dream recall
- creative breakthroughs
- tip-of-the-tongue resolution

## Accessibility variables

Potential factors affecting transitions:

| Class | Variables |
|---|---|
| Biological | sleep, fatigue, circadian rhythm, hormones, illness, nutrition |
| Cognitive | attention, priming, working memory, context, pattern similarity |
| Emotional | safety, fear, urgency, attachment, grief, stress |
| Environmental | location, sound, smell, objects, people, time |
| Unknown | mechanisms not yet characterized |

## Failure modes

### Stored but inaccessible

Information exists in representation but cannot be retrieved.

### Accessible but not expressible

The system can use the information but cannot explain it.

### Expressible but not applied

The system can state the information but does not act on it.

### Applied without explanation

The system acts on information before conscious explanation is available.

### False accessibility

Information feels accessible or certain but is inaccurate.

## Measurement questions

For each case, record:

1. What information became available?
2. What state did it appear to transition from?
3. What changed immediately before access?
4. Was it consciously available, expressible, applied, or only felt?
5. Did the information prove accurate, useful, partial, or false?
6. What alternative explanations remain plausible?

## Current strongest model statement

Information accessibility is a dynamic state property. It changes across time based on biological, cognitive, emotional, environmental, and potentially unknown conditions.
