# ASO-Lang Examples v0.1

## ASO derivation examples

```text
Existence + Difference DERIVES Boundary
Interaction + OrderedChange DERIVES Process
Difference + Interaction + OrderedChange DERIVES Information
Process + Information + OrderedChange DERIVES Feedback
Feedback + Persistence + RecursiveDesign DERIVES Adaptation
```

## IAT accessibility examples

```text
RepresentedInformation CAN_BE LatentInformation
LatentInformation TRANSITIONS_TO AccessibleInformation
AccessibleInformation TRANSITIONS_TO ConsciouslyAvailableInformation
ConsciouslyAvailableInformation TRANSITIONS_TO ExpressedInformation
```

## Invalid examples

```text
AI IS Intelligence
Biology IS Adaptation
Intuition PROVES UnknownMechanism
DejaVu PROVES PastLifeMemory
```

Valid alternatives:

```text
AI.System IMPLEMENTS AdaptiveFunctions
Biology.Mechanisms IMPLEMENTS Adaptation
Intuition EVIDENCES AccessibilityEvent
IF PastLifeMemoryHypothesis THEN DejaVuCase
```
