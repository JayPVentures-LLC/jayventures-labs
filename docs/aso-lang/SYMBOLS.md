# ASO-Lang Symbols v0.1

## Purpose

This file defines draft symbols for ASO and companion frameworks.

Symbols are provisional until ASO-Math begins.

## Primitive symbols

| Symbol | Term |
|---|---|
| E | Existence |
| Delta | Difference |
| I | Interaction |
| Omega | Ordered Change |

## Core object symbols

| Symbol | Term |
|---|---|
| Sys | System |
| s_t | State at sequence position t |
| P | Process |
| Inf | Information |
| Rep | Representation |
| F | Feedback |
| A | Adaptation |
| Ref | Reflection |
| D | Recursive Design |

## IAT symbols

| Symbol | Term |
|---|---|
| Acc | Accessibility |
| LatInf | Latent Information |
| AccInf | Accessible Information |
| ConInf | Consciously Available Information |
| ExpInf | Expressed Information |
| AppInf | Applied Information |

## Notation caution

This file avoids relying on special Unicode symbols as the only representation because repository, terminal, and tool contexts may vary.

When mathematical notation is needed later, ASO-Math may introduce LaTeX equivalents.
