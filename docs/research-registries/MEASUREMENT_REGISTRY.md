# Measurement Registry

Status: v0.1 operating draft

## Purpose

The measurement registry tracks constructs that require operational definitions, rubrics, reliability testing, validity testing, and calibration.

## Registry

| ID | Construct | Framework | Status | Measurement Need |
|---|---|---|---|---|
| ASO-MEAS-0001 | Adaptive Capacity | ASO | Draft target | Score information, representation, evaluation, execution, feedback, adaptation, and recursion. |
| ASO-MEAS-0002 | Representation Accuracy | ASO/IAT | Draft target | Compare represented information to validated external or internal reference. |
| ASO-MEAS-0003 | Feedback Delay | ASO | Draft target | Measure time between action outcome and system uptake. |
| ASO-MEAS-0004 | Persistence | ASO | Draft target | Measure continuity of identifiable structure across change. |
| ASO-MEAS-0005 | Recursive Capacity | ASO | Draft target | Measure ability to model and modify adaptation processes. |
| HUMAN-MEAS-0001 | Determination | Human Adaptive Architecture | Proposal target | Measure sustained intentional effort under obstacle conditions. |
| HUMAN-MEAS-0002 | Preservation Priority | Human Adaptive Architecture | Proposal target | Measure perceived protection/continuity priorities driving behavior. |
| HUMAN-MEAS-0003 | Threat Assessment | Human Adaptive Architecture | Proposal target | Measure detection, weighting, and response to perceived threat. |
| IAT-MEAS-0001 | Accessibility State | IAT | Draft target | Classify information availability state and transition. |
| IAT-MEAS-0002 | Intuition Intensity | IAT | Draft target | Rate subjective force of accessibility event. |
| IAT-MEAS-0003 | Intuition Calibration | IAT | Draft target | Compare confidence ratings to outcomes in prospective logs. |
| IAT-MEAS-0004 | Alternative Explanation Strength | IAT | Draft target | Rate sufficiency of ordinary explanations. |

## Required measurement artifact

Each measurement item must eventually include:

- construct definition;
- observable indicators;
- scoring procedure;
- evaluator instructions;
- reliability protocol;
- validity protocol;
- limitations;
- examples;
- non-examples;
- ethical boundaries if human-related.
