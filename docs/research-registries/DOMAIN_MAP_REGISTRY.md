# Domain Map Registry

Status: v0.1 operating draft

## Purpose

The domain map registry tracks fields that need translation into ASO, IAT, or future companion-framework language.

The goal is comparison without erasing domain-specific mechanisms.

## Required domain map

Each domain map should include:

- native terminology;
- ASO/IAT translation;
- function/mechanism distinction;
- measurable variables;
- known evidence;
- open questions;
- prediction candidates;
- domain-specific cautions.
