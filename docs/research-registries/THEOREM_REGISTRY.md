# Theorem Registry

Status: v0.1 operating draft

## Purpose

The theorem registry tracks logical claims that should be derivable from framework definitions and primitives.

Theorems are not empirical hypotheses. They require logical proof, not observational confirmation.

## Required proof record

Each theorem must eventually include:

- formal statement;
- required definitions;
- assumptions;
- proof sketch;
- counterexample attempt;
- status;
- version introduced;
- version revised.
