# JayPVentures Wix Implementation Checklist (Fast)

## Site Theme
- Set background = #0B0B0F
- Set primary text = #F5F5F7
- Set accent = #D10F2F

## Buttons
- Preset 1: Red solid (primary)
- Preset 2: White outline (secondary)
- Preset 3: Gradient outline (creator)

## Sections
- Spacing: 90-130px vertical padding
- Max width: 1100-1200px

## Hover Effects
- Apply glow only on creator buttons
- Apply glow only on creator cards
- Apply glow only on blog cards

## Global Checks
- Use iridescent only in button borders, hover effects, section dividers, blog cards
- Red is rare but powerful (one primary CTA per section)
