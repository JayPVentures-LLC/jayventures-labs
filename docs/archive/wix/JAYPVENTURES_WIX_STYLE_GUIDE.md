# JAYPVENTURES LLC × jaypventures

## Visual Style Guide Layout Map (Wix Build Blueprint)

### 0) Master Design Identity
Luxury governance infrastructure + futuristic creative lab energy.

## 1) Global Theme Map (Applies Everywhere)

### Global Colors (Roles)
- Base Black = primary background
- Pure White = primary text
- Authority Red = CTA + emphasis
- Iridescent Accent = innovation aura (hover + borders only)

### Global Spacing
- Large vertical padding between sections
- Minimal clutter
- Max 2-3 lines of text per block

### Global Motion
- Fade-in on scroll (subtle)
- Hover glow only
- No bouncing, no spinning

## 2) Header Map (All Pages)

### Desktop Header Layout
- Left: Logo / Text
- Right: Nav links

### Logo Style
- JAYPVENTURES LLC (white)

### Nav Links
- Home
- Services
- Trust

- Contact

- Red button: "Start a Project"

# Style Tile (Exact Design System)

## 1) Color Palette (Exact Hex)

### Core (JayPVentures LLC)
- Base Black: #0B0B0F
- True Black (for contrast blocks): #000000
- Soft White: #F5F5F7
- Pure White (rare, for highlights): #FFFFFF
- Authority Red: #D10F2F

### Neutral Grays (Luxury Depth)
- Graphite: #16161D
- Slate: #23232D
- Divider Gray: #2E2E3A
- Muted Text Gray: #B8B8C4

### Iridescent Accent (jaypventures creator brand)
- Iridescent Cyan: #4DEBFF
- Iridescent Violet: #8B5CFF
- Iridescent Magenta: #FF3DF0
- Iridescent Lime (very sparingly): #B6FF4D

## 2) Gradients (Exact, Wix-friendly)

### Creator Iridescent Border Gradient
- linear-gradient(90deg, #4DEBFF, #8B5CFF, #FF3DF0)

### Subtle Innovation Aura Overlay
- radial-gradient(circle at 20% 20%, rgba(77,235,255,0.18), transparent 50%),
- radial-gradient(circle at 80% 30%, rgba(255,61,240,0.14), transparent 55%),
- radial-gradient(circle at 50% 80%, rgba(139,92,255,0.12), transparent 60%)

## 3) Typography (Luxury + Modern)

### Headings (choose one)
- Option A (best): Helvetica Neue / Helvetica
- Option B: Montserrat
- Option C: Poppins

### Body Text (choose one)
- Option A: Inter
- Option B: Open Sans
- Option C: Helvetica / Arial

### Typography Rules
- Headings: bold
- Body: regular
- Line spacing: 1.4-1.6
- Max paragraph width: 650-750px
- No long paragraphs

## 4) Button Styles (Exact Specs)

### Primary Button (JayPVentures LLC CTA)
- Background: #D10F2F
- Text: #FFFFFF
- Radius: 10-14px
- Padding: 14px vertical / 22px horizontal
- Hover: darken slightly (#B90D29)

### Secondary Button (LLC Outline)
- Background: transparent
- Border: 1px solid #FFFFFF
- Text: #FFFFFF
- Hover: border turns red + slight glow

### Creator Button (jaypventures)
- Background: transparent
- Border: 2px gradient (iridescent gradient)
- Text: #FFFFFF
- Hover: soft glow
- Glow example: 0 0 18px rgba(77,235,255,0.25)

## 5) Card Styles (Services / Blog / Packages)

### Enterprise Cards
- Background: #16161D
- Border: 1px solid #2E2E3A
- Top accent line: 3px red #D10F2F
- Radius: 18-22px
- Shadow: very soft
- Hover: border slightly brighter gray, lift 2-4px

### Creator Cards
- Same base as enterprise
- Border glow uses gradient
- Hover glow appears

## 6) Dividers + Lines (Luxury Detail)

### Standard Divider
- 1px line, rgba(255,255,255,0.12)

### Innovation Divider
- 2px gradient line
- Opacity 40-60%

## 7) Icon Style (Advanced + Clean)
- Thin line icons only
- No filled cartoon icons
- No emoji icons
- Icon color: white at 80% opacity
- Red only for emphasis

## 8) Motion Style (Premium, not playful)

### Allowed
- Fade-in on scroll
- Hover glow
- Hover lift

### Forbidden
- Bounce
- Spin
- Neon flashing
- Parallax overload

## 9) Page Style Tiles (Per Page)

### Home
- 80% LLC
- 20% creator aura
- Use iridescent only in hero accents + creator lane

### Services
- 70% LLC
- 30% creator accents
- Enterprise section = red
- Creator section = gradient

### Trust
- 100% LLC
- 0% creator
- Only subtle gray dividers and red underline on hover

### Blog
- 50/50 blend
- Creator aura becomes visible
- Cards can glow
- Gradients show up

### Contact
- 95% LLC
- 5% creator
- Keep it clean
